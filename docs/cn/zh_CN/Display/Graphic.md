# Graphic绘图
__________________________
#### 功能说明
>除了几种可以直接使用的几何图形，还可以使用Graphic绘制一些复杂的图形

><img src="/image/Display/Graphic_Block.jpg"/>

>* __LCD clear__
清除屏幕内容
>* __LCD fill__
设置屏幕底色
>* __LCD print x,y__
在屏幕指定位置打印文本
>* __Font__
设置文本的字体
>* __LCD pixel__
在屏幕指定位置绘制一个像素点
>* __line__
绘制直线
>* __rect__
绘制矩形

#### 注意
>M5GO的屏幕分辨率为320X240


#### 使用案例
>使用Graphic功能，绘制一个三角形,并修改颜色为蓝色
