# Subscribe订阅
__________________________

#### 功能说明

>Subscribe订阅指的是，通讯中接收数据的环节，当发布者发布了信息后，订阅者将自动接收已订阅的主题（topic），消息内容（msg）

<!-- >![Repeat](/image/MQTT/Publish.jpg) -->

>* __Subscribe “主题”（topic）__
设定要订阅的主题

>* __Get topic data “内容”（msg）__
获取该订阅下的消息内容

#### 使用方法

>获取PASS主题下的消息内容，并显示到屏幕上
