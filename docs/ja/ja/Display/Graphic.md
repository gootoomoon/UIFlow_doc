# Graphic図
__________________________

#### 機能説明

>直接配置する図形に加えて、Graphicブロックを使用するとより複雑な図形を描画できます。

><img src="/image/Display/Graphic_Block.jpg"/>

>* __LCD clear__
スクリーンの内容をクリアします。

>* __LCD fill__
スクリーンを指定した色で塗りつぶします。

>* __LCD print x,y__
スクリーンの指定の位置にテキストを表示します。

>* __Font__
フォントを設定します。

>* __LCD pixel__
指定の位置にピクセルを描画します。

>* __line__
指定の位置に線を描画します。

>* __rect__
指定の位置に四角形を描画します。

#### 注意
>M5GOのスクリーンの解像度は横320x縦240です。


#### 使用方法

>Graphicブロックを使用して、三角形を描画し、色を青色に変更します。
