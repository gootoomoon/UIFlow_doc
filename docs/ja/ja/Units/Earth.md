# Earth
__________________________

#### 機能説明

>Earthモジュールは土壌の水分量検出モジュールです。テストピンを土壌に挿しこみ使用します。湿度データをアナログ量またはデジタル量でM5GOに送信することができます。

#### Block

>![Earth_Block](/image/Units/Earth_Block.jpg)

>* __Analog Value__
湿度をアナログ値で取得します。

>* __Digital Value__
湿度をデジタル値で取得します。閾値はUnit側で設定します。

#### 使用方法

>土壌中の湿度が15の時、音でお知らせします。
